# Proyecto: Arreglos Unidimensionales en Java

# Objetivo del Proyecto
Desarrollar un programa en Java que permita:
- Crear dos arreglos unidimensionales con 7 números ingresados por el usuario.
- Generar un tercer arreglo con la diferencia de los dos primeros.
- Calcular el promedio de todos los datos.
- Mostrar los resultados en pantalla.

# Instrucciones de Ejecución

1. Abrir el proyecto en un IDE como Visual Studio Code o IntelliJ.
2. Compilar el archivo principal.
3. Ejecutar el programa.
4. Ingresar los números solicitados por consola.
